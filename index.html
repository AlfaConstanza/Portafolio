Currículum Vitae

Desafío 1



Unidad Nº1

Objetivos de Aprendizaje

  • Crear un página web utilizando HTML.
  • Manejar las etiquetas p, h1, h2, h3, a, img, title y meta.
  • Implementar la estructura assets del proyecto revisada en la lectura.
  • Emplear adecuadamente la estructura y sintaxis de las etiquetas de un documento HTML, para dar solución a una problemática.
Landing Page

Desafío 2



Unidad Nº2

Objetivos de Aprendizaje

  • Crear un página web que contenga una imagen de fondo y un texto encima,con 3 imágenes de productos con su descripción .
  • Utilizar correctamente las etiquetas semánticas HTML.
  • Cargar el CSS desde un archivo css externo, desde una subcarpeta css dentro de la subcarpeta assets.
Iguana Page

Desafío 3



Unidad Nº3

Objetivos de Aprendizaje

  • Estructura del proyecto, debe existir un árbol de archivos.
  • Utilizar las propiedades display o float.
  • Implementar iconos de Font Awesome.
  • Seguir la maqueta, respetando fuentes, colores y tamaños
  • Aplicar etiquetas semánticas.
Cuppon Page

Desafío 4



Unidad Nº4

Objetivos de Aprendizaje

  • Seguir la maqueta del proyecto entregada.
  • Utilizar de Bootstrap: clases colores, margin -padding y componente Card.
  • Estructura del proyecto, debe existir un árbol de archivos.
  • Proyecto debidamente indentado y ordenado.
Github

Desafío 5



Unidad Nº5

Objetivos de Aprendizaje

  • Hacer un fork y clonar al siguiente repositorio https://github.com/gsanchezd/fdsw-github .
  • Modificar el proyecto clonado agregando datos propios.
  • Aprender a utilizar comandos Git, para el desarrollo del desafío.
  • Hacer por lo menos 5 commits con modificaciones al repositorio forkeado.
Portafolio

Desafío 6



Unidad Nº6

Objetivos de Aprendizaje

  • Construir un portafolio utilizando por lo menos 3 componentes y 5 clases de Bootstrap.
  • Desplegar el proyecto en Github Pages, en donde el portafolio sea la página principal.
  • El portafolio muestra los desafíos anteriores con imágenes y enlaces a las páginas correspondientes.
  • Forkear proyectos de compañeros, realizando modificaciones en estos.
